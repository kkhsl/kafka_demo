Form 表单提交

HTML 中的
元素可以用来提交用户输入的数据到后端服务。通过设置表单的 action 属性和 method 属性，可以将表单数据发送到后端服务端点。

iframe

使用

WebSocket

WebSocket 提供了浏览器与服务器之间全双工通信的能力，可以实时地传输数据，而不需要像 Ajax 那样进行轮询。

Server-Sent Events (SSE)

Server-Sent Events 允许服务器端单向推送数据到浏览器，而不需要浏览器发起请求。这在需要实时更新的应用中很有用。

Web Storage

HTML5 提供了本地存储（LocalStorage 和 SessionStorage）的功能，可以在客户端存储数据，并且不需要与服务器进行直接的通信

Fetch API

Fetch API 是用来进行网络请求和获取数据的现代 JavaScript API，替代了传统的 XMLHttpRequest。它提供了更强大和灵活的方式来处理网络请求
